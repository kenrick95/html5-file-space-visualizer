Chart Help

The Chart heading displays the current visualization displayed. Choose from a variety of visualizations using the icons along the top of the page.

  • Summary -- Overview and summary stats of your dataset
  • Visual Tree -- A 2d visual layout using circles and lines to represent the hierarchy.
  • Hierarchical List -- A table layout with summary data
  • Flattend List -- A table layout with summary data (Sortable)
  • Sunburst -- A circular layout with folders represented as arcs on concentric circles.
  • Tree Map -- A rectangle devided into smaller rectangles. The area of each rectangle represents the size of each folder.
  • Bubble Chart -- An unnested set of circles in which the area of the circle represents the size of the folder.
